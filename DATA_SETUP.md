# Dataset setup

The datasets are not included in this repository. Download them from the
original sources and place the files in the folders below, keeping the
original file names (do not rename).

## C-MAPSS (NASA turbofan)
Put the files in: `data/CMAPSSData/`
```
train_FD001.txt  test_FD001.txt  RUL_FD001.txt
train_FD002.txt  test_FD002.txt  RUL_FD002.txt
train_FD003.txt  test_FD003.txt  RUL_FD003.txt
train_FD004.txt  test_FD004.txt  RUL_FD004.txt
```

## Azure Predictive Maintenance
Put the files in: `data/azure/`
```
PdM_telemetry.csv
PdM_maint.csv
PdM_failures.csv
```

## NASA Battery (BatteryAgingARC)
Put the files in: `data/battery/`
```
B0005.mat
B0006.mat
B0007.mat
B0018.mat
```
